# 每日提醒版｜精華條列

## 🌱 今日核心觀念
- 有思考 = 有學習
- 有連結技術 = 沒有浪費時間
- 穩定 60 分 > 爆衝 100 分

---

## 🌤 三種日常節奏（依狀態彈性選擇）

### 🔥 前進日（有精神 / 想成長）
可以做：
- 寫 side project
- 解一個技術問題
- 重構一小段程式
- 寫技術筆記
- 深入閱讀一篇技術文章

目標：推進「能力邊界」

---

### 🌊 維持日（普通疲勞 / 想維持手感）
可以做：
- 看技術文章或文件
- 整理專案結構或註解
- 思考架構設計
- 看優質開源專案
- 回顧過去學過的東西

目標：保持與技術的連結

---

### 🌙 休息日（很累 / 心累 / 想放空）
可以做：
- 完全不碰技術
- 看影片、散步、發呆
- 睡覺、聽音樂
- 做讓自己安心的事

提醒：
> 休息不是浪費，是為了走更遠

---

## 🧠 心態校正
- 不碰技術 ≠ 浪費人生
- 疲勞 ≠ 不夠努力
- 慢慢來 ≠ 退步

真正的成長是：
> 持續很多年，而不是撐幾天

---

## 🌟 每日三問
1. 我今天有保持與目標的連結嗎？
2. 我今天有讓自己更穩定一點嗎？
3. 我今天有對自己溫柔一點嗎？
